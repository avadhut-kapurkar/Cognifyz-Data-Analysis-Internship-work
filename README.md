# Cognifyz Data Analysis Internship

Data Analysis Internship tasks completed using Python, Pandas, Matplotlib, and Google Colab.

## Overview

This repository contains the tasks completed during the Cognifyz Technologies Data Analysis Internship. The project focuses on data cleaning, analysis, and visualization using real-world restaurant datasets.

## Development Environment

All tasks were performed using Google Colab. Python libraries such as Pandas and Matplotlib were used for data analysis and visualization.

The notebooks in this repository can be downloaded and executed directly in Google Colab.

## Tools Used

* Python
* Pandas
* Matplotlib
* Google Colab
* GitHub

## Tasks Completed

### Level 1

* Top Cuisines Analysis
* City Analysis
* Price Range Distribution
* Online Delivery Analysis

### Level 2

* Restaurant Ratings Analysis
* Cuisine Combination Analysis
* Geographic Analysis
* Restaurant Chains Analysis

### Level 3

* Votes Analysis
* Price Range vs Services Analysis

## Project Execution

1. Upload the dataset to Google Colab.
2. Open the required notebook.
3. Run all cells to reproduce the analysis and visualizations.
4. Review the generated insights and conclusions.
